# AI Agents

Three Claude-powered agents that demonstrate AI-assisted engineering across the SDLC.

## Prerequisites

```bash
pip install -r agents/requirements.txt
export ANTHROPIC_API_KEY=sk-ant-...   # Windows: set ANTHROPIC_API_KEY=sk-ant-...
```

## Agents

### Design Agent
Reads the project spec and source files. Produces a design document with Mermaid architecture diagrams, sequence diagrams, ER diagrams, API contract, and design decisions.

```bash
python agents/design_agent.py
# Output: agents/output/design_output.md
```

### Development Agent
Reads all service source files. Audits error handling, logging coverage, observability gaps, and produces git commit message suggestions and top improvement recommendations.

```bash
python agents/dev_agent.py
# Output: agents/output/dev_output.md
```

### QA Agent
Reads source code and existing test suites. Produces coverage analysis, missing test scenarios, 5 new runnable pytest test functions, a functional test plan table, and a QA risk assessment.

```bash
python agents/qa_agent.py
# Output: agents/output/qa_output.md
```

## Output files

All agent outputs are saved to `agents/output/`. Pre-generated outputs are committed to the repository so reviewers can see the results without needing an API key.

| File | Generated by |
|---|---|
| `output/design_output.md` | Design Agent |
| `output/dev_output.md` | Development Agent |
| `output/qa_output.md` | QA Agent |
